Barcode Scanner+ usa la cámara de tu dispositivo para leer códigos de barras de todo tipo. Tras escanear el código de barras encuentras información acerca de productos, por ejemplo, puedes descubrir precios y opiniones sobre ellos. Igualmente, al escanear códigos de barras más modernos de dos dimensiones, como los códigos QR, puedes acceder a datos de contacto, marcadores, enlaces web y mucho más.
